🆘 QUICK TROUBLESHOOTING

Issue: 503 Bad Gateway
→ Check: sudo systemctl status ruangdengar
→ Restart: sudo systemctl restart ruangdengar

Issue: Static files 404
→ Run: python manage.py collectstatic --noinput
→ Fix: sudo chown -R www-data:www-data staticfiles/

Issue: Email not sending
→ Check: .env has EMAIL_HOST_USER and EMAIL_HOST_PASSWORD
→ Verify: PGPASSWORD=... psql -h ... test connection

Issue: Database connection error
→ Check: PostgreSQL running (sudo systemctl status postgresql)
→ Verify: .env has correct DB credentials

See REDEPLOY.md or DEPLOYMENT_CHECKLIST.md for more troubleshooting
