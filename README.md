# Ruang Dengar: Platform Digital Aman untuk Pelaporan Kekerasan Seksual

**"Dengarkan setiap suara, lindungi setiap cerita."** [cite: 3]

Ruang Dengar adalah platform digital berbasis web yang dikembangkan khusus untuk implementasi di **Telkom University Purwokerto**[cite: 62]. [cite_start]Platform ini berfungsi sebagai sarana pelaporan kekerasan seksual yang aman, anonim, dan terintegrasi dengan alur penanganan sesuai regulasi PPKS[cite: 63].

---

## 🚀 Fitur Utama

Platform ini dirancang dengan pendekatan *Trauma-Informed Design* untuk memberikan rasa aman kepada penyintas[cite: 258]:

* **Pelaporan Anonim & Terdaftar:** Memberikan opsi kepada korban untuk melapor tanpa mengungkap identitas di antarmuka admin[cite: 1052, 1054].
* **AI-Powered Moderation:** Menggunakan **Google Gemini API** untuk menganalisis narasi laporan, menentukan kategori, dan tingkat urgensi secara otomatis[cite: 287, 342].
* **Tracking & Status Real-time:** Pelapor dapat memantau progres penanganan kasus melalui timeline yang transparan[cite: 1081].
* **Booking Konseling:** Integrasi jadwal konselor kampus untuk mempermudah akses layanan psikologis[cite: 1100].
* **Dashboard Admin & Analitik:** Pusat komando bagi Satgas PPKS untuk manajemen kasus dan visualisasi data statistik[cite: 1138, 1289].

---

## 🛠️ Teknologi yang Digunakan

Proyek ini dibangun menggunakan *web application stack* yang modern dan aman[cite: 285]:

* **Backend:** Python 3.10 & Django 5.2 
* **Frontend:** HTML5, Tailwind CSS, & Django Templates 
* **Database:** PostgreSQL 14 (Production) [cite: 290]
* **AI Integration:** Google Gemini API 
* **Autentikasi:** Microsoft Entra ID (SSO Institusi) 
* **Deployment:** Gunicorn & Nginx [cite: 290]

---

## 🏗️ Perancangan Sistem

Sistem ini mengikuti metodologi **Agile Scrum** dan arsitektur **MVC**[cite: 239, 253]:

* **UML Diagrams:** Dilengkapi dengan Use Case, Class, Activity, dan Sequence Diagram untuk mendokumentasikan alur logika sistem[cite: 254].
* **Database Schema:** Perancangan ERD yang mencakup manajemen user (role-based), laporan, bukti, hingga rekam medis konseling[cite: 257, 949].
